The implementation uses proper transactional behavior:

```typescript
// If there are validation errors, don't import anything
if (errors.length > 0) {
  return res.status(400).json({
    error: 'Validation failed',
    errors,
    imported: 0,  // ← Zero records imported
  });
}

// Add all created sorties to the array
// ← This line only executes if errors.length === 0

```

This ensures that the sorties.push() operation only happens when there are
ZERO validation errors, preventing any partial imports.
